# Trouwuitnodiging - Bohemian/Vintage

Dit is een startproject voor een interactieve trouwuitnodiging in bohemian/vintage stijl, gemaakt met React, Framer Motion en Styled Components.

## Features
- 1 pagina met 7 secties (voor elke uitnodigingsstap)
- Parallax- en scroll-animaties (Framer Motion)
- Bohemian/vintage sfeer (aarde-tinten, aquarel, sierlijke fonts)
- Makkelijk uit te breiden

## Installatie
1. Installeer Node.js (https://nodejs.org/)
2. Open een terminal in deze map en voer uit:
   ```
   npm install
   npm start
   ```
3. De site draait op http://localhost:3000

## Structuur
- `src/App.js` — hoofdstructuur en secties
- `src/components/` — losse secties/components
- `src/styles/` — globale stijlen

## Uitbreiden
- Voeg per sectie een component toe in `src/components/`
- Gebruik Framer Motion voor animaties
- Pas kleuren/fonts aan in `src/styles/GlobalStyle.js`

Veel plezier met personaliseren!
